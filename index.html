To this page welcome!
Do you want to see how it is difficult to translate human languages? Even the Grand Master have problems keeping up with the meaning. Try YodaTalk, and see for yourself. Just choose different languages, so the translation will be more Yoda-talk-like. As you can see now the default translation sequence is: English → Japanese → Swedish → English. Be sure to try out other combinations of languages. YodaTalk is language-independent - it works not only with English language. It works in other languages, just because the algorithms we use for translations are not that good yet. These algorithms make mistakes. Be sure to try different input/output languages if you are a multilingual person. As Master once said: “In a dark place we find ourselves, and a little more knowledge lights our way.”
